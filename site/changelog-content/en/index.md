## Versioning

This project follows [Semantic Versioning](https://semver.org/):

- **Major (X.0.0)**: Breaking changes or major features
- **Minor (x.Y.0)**: New features, backward compatible
- **Patch (x.y.Z)**: Bug fixes, backward compatible
