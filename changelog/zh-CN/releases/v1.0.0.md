# Release v1.0.0

**发布日期**: 2025-02-13  
**版本类型**: 初始发布（主版本）

---

## 🎉 概览

v1.0.0 是 N-Body 粒子仿真系统的初始版本，提供完整的、生产就绪的大规模粒子仿真框架，支持在 NVIDIA GPU 上进行实时可视化。

### 主要特性

| 特性 | 描述 |
|------|------|
| 🚀 GPU 加速 | 基于 CUDA 的并行计算，支持数百万粒子 |
| 🔬 多种算法 | Direct N²、Barnes-Hut O(N log N)、Spatial Hash O(N) |
| 🎨 实时可视化 | 零拷贝 CUDA-OpenGL 互操作，60+ FPS |
| ⚛️ 能量守恒 | 辛 Velocity Verlet 积分器 |
| 🧪 全面测试 | Google Test + RapidCheck 基于属性的测试 |

---

## ✨ 新增

### 力计算算法

#### Direct N² (O(N²))
- 精确成对力计算
- 共享内存瓦片提高 GPU 效率
- 可配置线程块大小
- 适用：N < 10,000 粒子

#### Barnes-Hut (O(N log N))
- 基于八叉树的空间分解
- Morton 码排序（Z 阶曲线）
- 可配置开角（θ）
- 适用：大规模引力仿真

#### Spatial Hash (O(N))
- 短程力的均匀网格
- 27 单元格邻居搜索
- 可配置单元格大小和截断半径
- 适用：分子动力学、SPH 流体仿真

### 时间积分

- **Velocity Verlet** 辛积分器
- 二阶精度
- 每步单次力计算
- 长期能量稳定性

### 渲染系统

- OpenGL 3.3+ 点精灵渲染
- CUDA-OpenGL 零拷贝互操作
- 实时相机控制（轨道、缩放）
- 基于深度的粒子着色

### 粒子分布

- **均匀**: 边界框内随机
- **球形**: 球内均匀
- **圆盘**: 带初始旋转的扁平圆盘（星系仿真）

### 项目基础设施

- CMake 3.18+ 构建系统
- Google Test 集成
- RapidCheck 基于属性的测试
- MIT 许可证
- `.gitignore` 包含 CUDA/OpenGL 规则
- `.editorconfig` 保证格式一致

## 📝 API 亮点

```cpp
#include "nbody/particle_system.hpp"

using namespace nbody;

// 配置并初始化
SimulationConfig config;
config.particle_count = 100000;
config.force_method = ForceMethod::BARNES_HUT;

ParticleSystem system;
system.initialize(config);

// 运行仿真
for (int i = 0; i < 1000; ++i) {
    system.update(0.001f);
}

// 保存状态
system.saveState("checkpoint.nbody");
```

## ⚠️ 已知限制

1. **需要 CUDA**: 没有 NVIDIA GPU 和 CUDA toolkit 无法构建
2. **单 GPU**: 尚未实现多 GPU 支持
3. **仅限长程力**: Spatial Hash 假设短程相互作用
4. **无碰撞检测**: 粒子可以相互穿过

## 📦 安装

### 前置要求

- CUDA Toolkit 11.0+
- CMake 3.18+
- OpenGL 3.3+
- GLFW、GLEW、GLM
- C++17 编译器

### 构建

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(nproc)
```
